MovieServise

Get info about movies from public databases

View project on GitHub

Introduction

Get information about movies using our library as a wrapper for the OMDb and TMDb services.

Want to see how it works? Watch our MovieServiceDemo on youTube

You can search a movie by its title. If there is more than one movie with a similar title in the database, you will get a list of movies and their release year. If you find the movie you wanted to get information about on the movie the list, you can now specifically request for the data of that movie by its title.

Audience

This library is intended for developers and techy movie lovers who want to get information about movies. You can find the full MovieService API documentation in our repository's wiki pages. Click the view project on github button on the top right to get to our git repository and then go to the click on the wiki label.
