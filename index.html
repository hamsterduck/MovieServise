Introduction

The Movie Service API is a service that sends requests to get information about movies in a database of your choice using either the RESTful API of Omdb or Tmdb.

This Video illustrates the use of the Movie Service API.

You can search a movie by its title. If there is more than one movie with a similar title in the database, you will get a list of movies and their release year. If you find the movie you wanted to get information about on the movie the list, you can now specifically request for the data of that movie.

Audience

This document is intended for developers and techy movie lovers who want to get information about movies. It provides an introduction to using Movie Service API.
